## 项目简介

**易经学堂**是一款专注于中国传统文化教育的iOS应用，致力于让用户系统学习易经知识，传承和弘扬传统文化。应用采用现代化的设计理念，结合游戏化学习方式，为国学爱好者提供便捷、有趣的学习工具。

### 核心特色

- ✅ **完整知识体系**：六十四卦详解、八卦基础知识、历史文化内容
- ✅ **互动学习**：卦象记忆游戏、知识问答，寓教于乐
- ✅ **学习工具**：笔记、收藏、进度追踪，个性化学习
- ✅ **离线使用**：所有数据本地存储，无需网络连接
- ✅ **精美设计**：现代简约风格，融合传统文化元素

## 产品定位

- **类型**：教育类应用
- **目标用户**：国学爱好者、易经初学者、传统文化研究者
- **核心价值**：系统化学习、文化传承、知识传播
- **差异化**：强调教育属性，避免占卜功能，专注知识学习

## 常见问题

### Q1: 为什么选择本地存储而不是云端？
**A:** 本地存储简化开发，降低成本，避免数据隐私问题，且符合离线学习场景。

### Q2: 如何避免被审核拒绝？
**A:** 强调教育属性，避免占卜功能，所有文案强调学习研究，提供完善的用户协议。

### Q3: 后期能否添加起卦功能？
**A:** 可以谨慎添加"模拟练习"功能，但必须定位为"学习演示"，保持教育框架。

### Q4: 数据如何更新？
**A:** MVP版本数据内置在应用中，后期可考虑支持iCloud同步或数据包更新。
