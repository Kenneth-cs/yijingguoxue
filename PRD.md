# 易经学堂 - 产品需求文档（PRD）

## 一、产品概述

### 1.1 产品定位
易经学堂是一款中国传统文化学习工具，专注于易经知识的教育传播，为国学爱好者提供系统化的学习平台。

### 1.2 产品目标
- 帮助用户系统学习易经知识
- 传承和弘扬中国传统文化
- 提供互动有趣的学习体验
- 顺利通过App Store审核

### 1.3 目标用户
- 国学爱好者
- 易经初学者
- 传统文化研究者
- 对中国哲学感兴趣的用户

### 1.4 核心价值
- **教育性**：系统化的知识体系
- **互动性**：游戏化学习体验
- **文化性**：传统文化传承
- **工具性**：便捷的学习工具

## 二、功能需求

### 2.1 核心功能模块

#### 模块一：易经知识百科 ⭐⭐⭐⭐⭐

**功能描述**
提供完整的易经知识体系，包括六十四卦详解。

**详细需求**

1. **六十四卦列表**
   - 展示所有64卦的卦象、卦名
   - 支持按序号、卦名搜索
   - 支持收藏功能
   - 卡片式布局，美观易读

2. **卦象详情页**
   - 卦象图示（上下卦组合）
   - 卦名、卦辞
   - 六爻爻辞（初爻到上爻）
   - 白话释义
   - 象传、彖传（可选）
   - 历史典故和应用案例

3. **八卦基础知识**
   - 八卦基本信息（乾、坤、震、巽、坎、离、艮、兑）
   - 八卦属性（五行、方位、象征等）
   - 配图说明

4. **搜索功能**
   - 按卦名搜索
   - 按关键词搜索内容

#### 模块二：互动学习 ⭐⭐⭐⭐

**功能描述**
通过游戏化方式增强学习效果。

**详细需求**

1. **卦象记忆游戏**
   - **认卦游戏**：给出卦象，选择正确卦名
   - **拼卦游戏**：根据上下卦拼出完整卦象
   - **难度分级**：简单/中等/困难
   - **积分系统**：答对得分，记录最高分
   - **学习模式**：显示正确答案和解释

2. **知识问答**
   - 选择题形式（单选）
   - 题目涵盖：卦象、卦辞、历史知识
   - 每次随机10题
   - 答题后显示得分和解析
   - 错题本功能

3. **每日一卦**
   - 每天推送一个卦象学习
   - 精简版卦象介绍
   - 支持标记已学习
   - 学习日历展示

#### 模块三：历史文化 ⭐⭐⭐

**功能描述**
提供易经相关的历史文化知识。

**详细需求**

1. **易经发展史**
   - 时间轴展示易经发展历程
   - 重要历史节点介绍
   - 配图和文字说明

2. **历代易学家**
   - 易学名家列表（孔子、王弼、程颐等）
   - 人物简介
   - 主要贡献和著作

3. **相关典籍**
   - 《周易》经传介绍
   - 相关经典书籍推荐
   - 外部链接（书籍购买等）

#### 模块四：学习工具 ⭐⭐⭐⭐

**功能描述**
辅助用户更好地学习和管理进度。

**详细需求**

1. **学习笔记**
   - 在任意卦象详情页添加笔记
   - 支持文字记录
   - 笔记列表查看
   - 编辑和删除功能

2. **收藏功能**
   - 收藏感兴趣的卦象
   - 收藏列表独立展示
   - 快速访问

3. **学习进度**
   - 已学习卦象统计
   - 学习天数统计
   - 学习时长统计（可选）
   - 进度可视化展示

4. **学习提醒**
   - 每日学习提醒推送
   - 可自定义提醒时间
   - 可开关提醒

### 2.2 辅助功能

#### 用户系统（简化版）
- 本地存储用户数据
- 无需注册登录
- 数据本地化

#### 设置页面
- 字体大小调节
- 夜间模式开关
- 通知设置
- 关于页面（版本信息、联系方式）
- 用户协议和隐私政策

## 四、页面结构

### 4.1 底部导航栏（Tab Bar）
1. **首页** - 每日一卦、学习统计、快捷入口
2. **百科** - 六十四卦列表
3. **练习** - 互动学习游戏
4. **我的** - 收藏、笔记、设置

### 4.2 主要页面流程

```
启动页
  ↓
首页
  ├─ 每日一卦 → 卦象详情
  ├─ 学习统计
  └─ 快捷入口 → 各功能模块

百科Tab
  ├─ 六十四卦列表 → 卦象详情
  ├─ 八卦基础 → 八卦详情
  └─ 搜索 → 搜索结果 → 卦象详情

练习Tab
  ├─ 卦象记忆游戏 → 游戏界面 → 结果页
  ├─ 知识问答 → 答题界面 → 结果页
  └─ 历史记录

我的Tab
  ├─ 我的收藏 → 卦象详情
  ├─ 我的笔记 → 笔记详情
  ├─ 学习进度
  └─ 设置
      ├─ 通用设置
      ├─ 通知设置
      ├─ 关于我们
      └─ 用户协议
```

## 五、非功能性需求

### 5.1 性能要求
- App启动时间 < 2秒
- 页面切换流畅（60fps）
- 列表滑动流畅
- 离线可用（所有数据本地存储）

### 5.2 兼容性要求
- iOS 14.0+
- 支持iPhone和iPad
- 适配各种屏幕尺寸
- 支持深色模式

### 5.3 数据要求
- 所有数据本地存储
- 无需网络连接即可使用
- 数据持久化保存
- 支持数据导出（可选）

### 5.4 用户体验要求
- 界面简洁清爽
- 操作直观易懂
- 提供操作反馈
- 加载状态提示

## 六、设计原则

### 6.1 视觉风格
- **风格**：现代简约 + 传统文化元素
- **色调**：以素雅的中国传统色为主（如月白、竹青、黛蓝）
- **字体**：系统字体，保证清晰可读
- **图标**：简洁线条图标

### 6.2 交互原则
- 符合iOS人机交互规范
- 操作步骤最小化
- 提供明确的视觉反馈
- 支持常见手势操作

### 6.3 内容原则
- **强调教育属性**：所有文案强调学习、研究、文化传承
- **避免敏感词汇**：不使用"占卜"、"算命"、"预测"等词
- **学术化表达**：使用"研究"、"学习"、"传统文化"等词汇
- **免责声明**：明确说明仅供学习研究使用

## 七、审核策略

### 7.1 应用描述要点
- 定位为教育类应用
- 强调传统文化学习
- 突出知识性和教育性
- 展示互动学习功能

### 7.2 关键词建议
- 易经、国学、传统文化、文化学习
- 避免：占卜、算命、运势、预测

### 7.3 应用分类
- 主分类：教育（Education）
- 次分类：参考（Reference）

### 7.4 用户协议要点
- 明确应用用于文化学习和研究
- 不提供任何形式的预测服务
- 不保证任何结果的准确性
- 内容仅供参考学习

## 八、数据内容规划

### 8.1 六十四卦数据结构
每个卦包含：
- 卦序（1-64）
- 卦名
- 上卦、下卦
- 卦象图（六个爻）
- 卦辞
- 六个爻辞
- 白话释义
- 简短说明
- 相关典故（可选）

### 8.2 内容来源
- 《周易》原文（公版内容）
- 白话翻译（自行整理）
- 历史知识（公开资料整理）

### 8.3 内容审核
- 所有内容经过审核
- 确保符合社会主义核心价值观
- 避免封建迷信内容
- 强调文化研究属性

## 十、成功指标

### 产品指标
- App Store审核通过
- 用户留存率 > 30%（次日）
- 平均使用时长 > 5分钟

## 十一、风险与应对

### 11.1 审核风险
**风险**：被认定为占卜类应用
**应对**：
- 强化教育属性
- 完善用户协议
- 准备申诉材料

### 11.2 内容风险
**风险**：内容被认为是封建迷信
**应对**：
- 内容强调学术研究
- 添加历史文化背景
- 避免功利性描述

### 11.3 竞争风险
**风险**：同类应用较多
**应对**：
- 专注教育定位
- 提升用户体验
- 持续内容更新
