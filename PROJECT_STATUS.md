## 📱 如何运行项目

### 环境要求
- macOS 12.0+
- Xcode 14.0+
- iOS 14.0+ 设备或模拟器

### 运行步骤

1. **打开项目**
```bash
cd "/Users/cs/Desktop/CS/AI/易经国学堂"
open 易经国学堂.xcodeproj
```

2. **选择模拟器**
   - 在Xcode顶部选择目标设备
   - 推荐：iPhone 14 Pro

3. **运行项目**
   - 点击 ▶️ 按钮或按 `Cmd + R`
   - 等待编译完成

### 已知运行问题
- ✅ 无编译错误
- ✅ 无运行时崩溃
- ⚠️ 部分数据显示"示例数据"
