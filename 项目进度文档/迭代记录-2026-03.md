## v1.1 · UI 视觉重构（2026-03-06）

### 目标
按照设计稿截图全面调整首页及全局视觉风格。

### 完成内容

#### 首页（HomeView）重构
- 导航栏：改为绿色居中标题"易经学院"
- 欢迎横幅：深绿渐变背景 + 太极图标 + 黄色装饰线 + 右侧半透明装饰圆
- 今日一卦：白色卡片 + 六爻图标 + 卦名格式"第X卦 XX" + 上下卦描述 + 绿色详情按钮
- 快速入口：4宫横排（浏览卦象 / 八卦基础 / 认卦游戏 / 我的收藏）
- 学习概览：3张并排白色统计卡片（已学卦象含进度条 / 我的收藏 / 连续学习）
- 整体背景：纯白

#### 全局字体
- 全部改为宋体 **Songti SC**，三档字重：
  - `Songti SC Bold` → 标题、重要数字
  - `Songti SC` → 正文、卦辞
  - `Songti SC Light` → 辅助说明、小标签

#### 底部 Tab 栏
- 背景：纯白
- 选中颜色：主题绿 `#0B6B52`
- 未选中颜色：中灰蓝 `#606878`
- 图标：替换为用户提供的自定义 PNG（模板模式，自动变色）
- 标签字体：宋体
