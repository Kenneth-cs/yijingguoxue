# 🎨 易经国学堂 - UI设计指南

## 📱 设计理念

基于易经文化和App Icon的墨绿色、金色主题，打造现代简约且富有文化底蕴的用户界面。

## 🎯 设计元素

### 1. 圆角半径
- **标准圆角**：16pt - 用于卡片、按钮
- **小圆角**：10pt - 用于小型元素
- **大圆角**：20pt - 用于大型容器

### 2. 间距系统
- **小间距**：8pt
- **标准间距**：16pt
- **大间距**：24pt
- **超大间距**：32pt

### 3. 阴影效果
```swift
标准阴影：
- 颜色：黑色 8% 透明度
- 半径：8pt
- 偏移：(0, 4)

强调阴影：
- 颜色：主色 30% 透明度
- 半径：10pt
- 偏移：(0, 5)
```

### 4. 渐变效果

**主色渐变**：
```swift
墨绿渐变：#1B5E4F → #2E8B6F
方向：左上到右下
```

**金色渐变**：
```swift
金色渐变：#D4AF37 → #E5C766
方向：左上到右下
```

**优雅渐变**（背景装饰）：
```swift
淡雅渐变：墨绿10%透明 → 金色10%透明
方向：左上到右下
```

## 📐 组件设计规范

### 首页（HomeView）

#### 1. 欢迎卡片
- **背景**：墨绿色渐变
- **高度**：140pt
- **装饰**：半透明八卦图案（右上角）
- **文字**：白色，带太极符号☯️
- **强调线**：金色，3pt高，60pt宽

#### 2. 每日一卦卡片
- **标题栏**：金色渐变背景
- **卦象区**：圆形装饰背景（80pt直径）
- **内容**：白色背景，墨绿色标题
- **指示器**：金色圆形箭头图标

#### 3. 学习进度卡片
- **进度条**：
  - 背景：淡雅渐变
  - 进度：墨绿色渐变
  - 高度：12pt，圆角10pt
- **统计项**：
  - 图标：圆形背景（44pt直径）
  - 数值：20pt粗体
  - 分隔线：垂直，50pt高

#### 4. 快速入口按钮
- **尺寸**：自适应宽度 × 110pt高
- **背景**：各色渐变
- **图标**：圆形半透明背景（50pt直径）
- **文字**：白色，14pt半粗体

### 百科页面（EncyclopediaView）

#### 1. 卦象卡片
- **卦象区**：
  - 圆角矩形背景（65×75pt）
  - 淡雅渐变填充
- **文字层次**：
  - 标题：17pt半粗体，墨绿色
  - 副标题：13pt，浅墨绿色
  - 描述：12pt，次要色
- **箭头**：金色半透明

#### 2. 八卦卡片
- **卦象区**：
  - 圆角矩形（70×70pt）
  - 淡雅渐变背景
  - 36pt符号
- **属性标签**：
  - 图标+文字组合
  - 12pt字体
  - 间距12pt

### 八卦详情页（TrigramDetailView）

#### 1. 顶部卡片
- **背景**：淡雅渐变
- **卦象**：120pt大小
- **卦名**：32pt粗体
- **二进制**：16pt，字间距8pt

#### 2. 属性行
- **图标**：
  - 圆形背景（40pt直径）
  - 彩色半透明填充
  - 16pt图标
- **布局**：
  - 图标 + 标题（50pt宽）+ 值
  - 分隔线：左侧缩进60pt

#### 3. 说明卡片
- **标题**：图标+文字，18pt半粗体
- **内容**：16pt，行间距6pt
- **内边距**：20pt

## 🎯 设计原则

### 1. 文化融合
- 传统色彩（墨绿、金色）与现代设计结合
- 易经元素（八卦、阴阳）作为视觉装饰
- 保持简洁，避免过度装饰

### 2. 层次清晰
- 使用色彩和字重建立视觉层次
- 重要信息使用主色强调
- 次要信息使用灰色弱化

### 3. 一致性
- 统一的圆角、间距、阴影
- 统一的色彩使用规范
- 统一的交互反馈

### 4. 可读性
- 充足的行间距和字间距
- 合适的字号（最小12pt）
- 高对比度的文字颜色

### 5. 优雅简约
- 留白充足
- 渐变使用克制
- 装饰元素点到为止
