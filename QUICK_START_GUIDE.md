# 易经国学堂 - 快速开始指南

## 🚀 快速启动

### 1. 环境要求
- macOS 13.0+
- Xcode 15.0+
- iOS 15.0+ 模拟器或设备

### 2. 打开项目
```bash
cd "/Users/cs/Desktop/CS/AI/易经国学堂"
open 易经国学堂.xcodeproj
```

### 3. 运行应用
1. 在 Xcode 中选择目标设备（模拟器或真机）
2. 按 `⌘ + R` 或点击运行按钮
3. 等待编译完成，应用将自动启动

## 📱 功能使用指南

### 首页
- 查看每日推荐卦象
- 查看学习进度统计
- 通过快速入口访问各个功能模块

### 易经百科
1. **浏览六十四卦**
   - 点击"易经百科"标签
   - 选择"六十四卦"选项卡
   - 浏览或搜索卦象
   - 点击卦象卡片查看详情

2. **学习八卦基础**
   - 选择"八卦"选项卡
   - 浏览八卦的详细信息

3. **收藏功能**
   - 在卦象详情页面点击右上角星标按钮
   - 在"我的收藏"中查看所有收藏的卦象

### 互动练习
1. **开始游戏**
   - 点击"练习"标签
   - 选择游戏类型：
     - 认卦游戏：识别卦象符号
     - 知识问答：测试易经知识
     - 卦象识别：识别卦名
     - 爻辞配对：匹配爻辞
   
2. **游戏玩法**
   - 阅读问题和提示
   - 选择答案选项
   - 提交答案查看结果
   - 完成所有题目后查看成绩

3. **查看记录**
   - 在练习页面底部查看最近记录
   - 点击"查看全部"查看完整游戏历史

### 学习笔记
1. **创建笔记**
   - 进入"我的" → "学习笔记"
   - 点击右上角"+"按钮
   - 选择关联的卦象
   - 输入笔记内容
   - 点击"保存"

2. **编辑笔记**
   - 在笔记列表中点击笔记
   - 点击右上角菜单按钮
   - 选择"编辑"
   - 修改内容后保存

3. **删除笔记**
   - 在笔记列表中左滑删除
   - 或在笔记详情页面选择"删除"

### 个人中心
- 查看学习统计数据
- 管理收藏和笔记
- 查看游戏记录
- 阅读关于信息和隐私政策

## 🎮 游戏说明

### 认卦游戏
- **目标**: 根据卦象符号识别卦名
- **玩法**: 观察卦象符号，从4个选项中选择正确的卦名
- **难度**: ⭐⭐⭐

### 知识问答
- **目标**: 测试易经基础知识
- **玩法**: 回答关于卦辞、卦名、卦象组成等问题
- **难度**: ⭐⭐⭐⭐

### 卦象识别
- **目标**: 根据二进制表示识别卦名
- **玩法**: 观察二进制卦象，选择对应的卦名
- **难度**: ⭐⭐⭐⭐⭐

### 爻辞配对
- **目标**: 将爻辞与对应的卦象匹配
- **玩法**: 阅读爻辞内容，选择正确的卦象
- **难度**: ⭐⭐⭐⭐

## 📚 学习建议

### 初学者路径
1. **第一周**: 了解八卦基础
   - 学习八卦的名称和象征
   - 理解五行属性和方位
   - 记住八卦符号

2. **第二周**: 学习重要卦象
   - 从乾卦、坤卦开始
   - 每天学习1-2个卦象
   - 记录学习笔记

3. **第三周**: 系统学习六十四卦
   - 按照顺序学习
   - 理解卦辞含义
   - 使用游戏巩固记忆

4. **第四周**: 深入研究
   - 学习爻辞
   - 理解卦象之间的关系
   - 应用到实际思考中

### 进阶用户建议
- 每日坚持学习一卦
- 使用笔记记录心得
- 定期复习已学内容
- 通过游戏测试掌握程度
- 关注卦象之间的联系

## 🔍 数据说明

### 六十四卦数据
每个卦象包含：
- **基本信息**: 卦序、卦名、完整名称
- **结构**: 上卦、下卦、二进制表示
- **经文**: 卦辞
- **释义**: 白话解释
- **爻辞**: 6条爻辞及其释义
- **符号**: Unicode卦象符号

### 八卦数据
每个八卦包含：
- **基本信息**: 名称、符号
- **属性**: 五行、方位、性质
- **象征**: 家庭关系、自然象征
- **说明**: 详细解释

## 🐛 常见问题

### Q: 数据会丢失吗？
A: 不会。所有数据都自动保存在本地，除非手动清除应用数据或卸载应用。

### Q: 可以导出笔记吗？
A: 当前版本暂不支持导出，后续版本会添加此功能。

### Q: 游戏记录保存多久？
A: 系统保留最近100条游戏记录。

### Q: 如何重置学习进度？
A: 在设置中可以找到"清空数据"选项（注意：此操作不可逆）。

### Q: 应用支持哪些iOS版本？
A: 支持 iOS 15.0 及以上版本。
