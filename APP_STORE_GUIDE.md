## 审核前准备

### 技术准备清单

- [ ] **Apple Developer账号**
  - 个人或公司账号（$99/年）
  - 完成实名认证
  - 支付信息完善

- [ ] **证书配置**
  - Distribution Certificate创建
  - App ID注册
  - Provisioning Profile配置

- [ ] **应用构建**
  - Release配置优化
  - 版本号设置（如1.0.0）
  - 构建号设置（如1）
  - Archive构建成功

- [ ] **测试验证**
  - 在真机上完整测试
  - 无崩溃、无明显bug
  - 所有功能正常工作
  - 性能表现良好

## 应用资源准备

### 1. 应用图标（App Icon）

**规格要求：**
- 尺寸：1024 × 1024 像素
- 格式：PNG（无透明通道）
- 色彩空间：RGB
- 无圆角（系统自动处理）

**设计建议：**
```
主题：易经文化元素
- 可使用八卦图案（简化版）
- 使用传统中国色
- 保持简洁，避免文字过多
- 确保在小尺寸下依然清晰
```

**示例设计方向：**
- 方案1：简化的八卦图案 + "易"字
- 方案2：书卷图案 + 传统纹样
- 方案3：卦象符号（⚊⚋）抽象设计

### 2. 应用截图（Screenshots）

**必需尺寸：**
- 6.7英寸显示屏（iPhone 14 Pro Max等）：1290 × 2796 像素
- 5.5英寸显示屏（iPhone 8 Plus等）：1242 × 2208 像素

**可选尺寸：**
- 6.5英寸：1284 × 2778（iPhone 13 Pro Max）
- 5.8英寸：1125 × 2436（iPhone X/XS）
- iPad Pro：2048 × 2732

**截图数量：**
- 最少：3张
- 最多：10张
- 建议：5-6张

**内容规划：**
```
第1张：首页 - 每日一卦展示
第2张：六十四卦列表页
第3张：卦象详情页
第4张：卦象记忆游戏
第5张：学习进度统计
第6张：知识问答功能（可选）
```

**设计要点：**
- 突出教育学习功能
- 界面美观，体现文化内涵
- 可添加标注文字说明
- 避免显示任何"占卜"相关内容

### 3. 应用预览视频（可选但推荐）

**规格：**
- 时长：15-30秒
- 分辨率：与截图尺寸一致
- 格式：M4V、MP4或MOV

**内容建议：**
```
0-5秒：展示应用图标和名称
5-15秒：快速浏览核心功能
15-25秒：展示互动学习场景
25-30秒：Slogan或应用定位
```

**文案示例：**
"易经学堂 - 系统学习传统文化"
"互动游戏，轻松记忆六十四卦"

## 元数据撰写

### 1. 应用名称（App Name）

**限制：**
- 最多30个字符（包括空格）

**建议：**
```
方案1：易经学堂
方案2：易经学习
方案3：国学易经
方案4：周易学堂

推荐：易经学堂（简洁明确）
```

**注意事项：**
- ❌ 避免：易经占卜、易经算命、易经预测
- ✅ 推荐：强调学习、教育、文化

### 2. 副标题（Subtitle）

**限制：**
- 最多30个字符

**建议：**
```
方案1：传统文化学习工具
方案2：系统学习六十四卦
方案3：国学知识互动学习
方案4：易经知识百科全书

推荐：传统文化学习工具
```

• 学习笔记记录

【联系我们】
如有任何问题或建议，请通过应用内反馈功能联系我们。

**关键要点：**
- ✅ 强调"学习"、"教育"、"文化传承"
- ✅ 突出系统化、知识性
- ✅ 说明教育价值和意义
- ❌ 避免"占卜"、"预测"、"算命"等词
- ❌ 不要暗示任何功利性用途

### 4. 关键词（Keywords）

**限制：**
- 最多100个字符
- 使用英文逗号分隔

**推荐关键词：**
```
易经,国学,传统文化,文化学习,周易,六十四卦,八卦,易经学习,国学经典,中国文化,哲学,教育,知识学习,文化传承,易学
```

**优化建议：**
- 避免重复词汇
- 不使用品牌词（如"中国最好的"）
- 使用用户常搜索的词
- 定期根据数据优化

### 5. 宣传文本（Promotional Text）

**限制：**
- 最多170个字符
- 可随时更新，无需新版本

**建议：**
```

```

## 审核策略

### 针对准则4.3(b)的应对

**问题：**
苹果可能认为应用属于"占卜类垃圾应用"

**应对策略：**

#### 1. 定位明确
```
✅ 我们是：教育类应用
❌ 我们不是：占卜工具
```

#### 2. 功能设计
```
✅ 包含：知识学习、互动练习、历史文化
❌ 避免：起卦功能、吉凶判断、个人预测
```

#### 3. 文案表述
```
✅ 使用：学习、研究、文化、传承
❌ 避免：占卜、算命、预测、运势
```

#### 4. 用户协议
在应用内明确声明：
```
本应用专注于易经知识的教育传播，
所有内容仅供文化学习和研究使用，
不提供任何形式的占卜或预测服务。
```

### 提交时的审核说明

在App Store Connect的"审核备注"中添加：

```
致审核团队：

易经学堂是一款专注于中国传统文化教育的应用。

【应用定位】
这是一款教育类应用，类似于历史百科或文化学习工具，
帮助用户系统学习《易经》这一中国传统经典的知识内容。

【核心功能】
1. 知识百科：提供六十四卦的详细介绍和历史文化背景
2. 互动学习：通过游戏化方式帮助记忆和理解
3. 学习工具：笔记、收藏、进度追踪等辅助学习

【与占卜类应用的区别】
• 不提供任何形式的占卜或预测功能
• 不涉及个人运势、吉凶判断
• 所有内容均为历史文化知识教育
• 明确声明仅供学习研究使用

【教育价值】
《易经》是中国传统文化的重要组成部分，联合国教科文组织
将其列入世界记忆名录。本应用旨在帮助用户了解和传承
这一优秀传统文化。

【参考对比】
类似定位的应用：诗词学习类、古文学习类、历史百科类

感谢审核！如有任何疑问，请随时联系。
```

## 常见拒绝原因及应对

### 拒绝原因1：4.3(b) - 垃圾应用（占卜类）

**苹果反馈：**
```
"您的应用主要功能是占卜，属于垃圾应用范畴..."
```

**应对措施：**

1. **立即修改应用描述**
   - 删除所有可能被误解为占卜的文案
   - 强化教育学习定位

2. **准备申诉信**（见下方模板）

3. **如有必要，调整功能**
   - 增加更多教育内容占比
   - 弱化可能引起误解的功能

### 拒绝原因2：2.1 - 性能问题

**苹果反馈：**
```
"应用在审核时崩溃或无法正常使用..."
```

**应对措施：**
- 在多个设备上彻底测试
- 检查数据文件完整性
- 确保网络不可用时也能正常使用
- 提供详细的使用说明

### 拒绝原因3：5.1.1 - 隐私政策

**苹果反馈：**
```
"隐私政策不完整或链接无效..."
```

**应对措施：**
- 确保隐私政策URL可访问
- 内容完整，涵盖数据收集说明
- 即使不收集数据也要明确说明

## 提交流程

### 1. 在Xcode中Archive

```bash
1. 选择 Generic iOS Device
2. Product → Archive
3. 等待构建完成
4. 在Organizer中查看
```

### 2. 上传到App Store Connect

```bash
1. 点击 "Distribute App"
2. 选择 "App Store Connect"
3. 选择 "Upload"
4. 等待处理（10-30分钟）
```

### 3. 在App Store Connect配置

```bash
1. 登录 https://appstoreconnect.apple.com
2. 选择应用
3. 填写所有必需信息
4. 上传截图和图标
5. 选择构建版本
6. 提交审核
```

### 4. 等待审核

```
• 审核时间：通常1-3天
• 状态查看：App Store Connect
• 邮件通知：审核结果
```

## 审核后处理

### 审核通过 ✅

1. **发布应用**
   - 可选择立即发布或手动发布
   - 设置发布日期（如需）

2. **监控数据**
   - 下载量
   - 崩溃率
   - 用户评分

3. **收集反馈**
   - App Store评论
   - 应用内反馈
   - 社交媒体

### 审核拒绝 ❌

1. **仔细阅读拒绝原因**
   - 理解具体问题
   - 查看截图（如有）

2. **制定应对方案**
   - 修改代码/内容
   - 准备申诉

3. **重新提交或申诉**
   - 修复问题后重新提交
   - 或在Resolution Center申诉
